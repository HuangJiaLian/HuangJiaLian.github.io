视频格式转换

# 无损转化mkv到mp4
for i in *.mkv; do ffmpeg -i "$i" -codec copy "${i%.*}.mp4"; done

# 快速转换LRV到mov
for i in *.LRV; do ffmpeg -i "$i" -codec copy "${i%.*}.mov"; done

# 视频转换成gif, 按比例缩放
ffmpeg -i cube.mov -r 5 -vf scale=640:-1 cube.gif

# 视频编辑

分辨率自定义设置 2560 × 1600 这样就是全屏幕了

快捷键A 选择

快捷键B ✂️

快捷键 cmd + C 同时可以赋值这个片段的属性

Option + G 合并多个片段

局部模糊:1. 剪刀留下需要片断 2. 高斯效果拖到片段里 3. 高斯效果里右边，添加形状遮罩。

添加文字: 1. 剪刀留下需要片断 2. Edit→Connect To Title→Basic Title 3. 右上角编辑文字。

快速播放: 1. Retime pop-up menu below the viewer 2. choose Slow or Fast

# 音频编辑

导出音频: 右上角 Share → Master File → Settings → Format[Audio Only] MP3 Format

静音片段: Choose Modify → Adjust Volume → Silence (–∞)

淡入淡出: Modify → Adjust Audio Fades → Fade In or Out

# 制作字幕

Final Cut Pro中识别的srt文件里面的标号不能从0开始，必须从1开始。

我尝试将网易见外生成的srt文件中的第0项合并到第1项，然后删除第0项后，便可成功导入Final Cut Pro了。

修改srt文件，使其编号从1开始，而不是从0.
