# 第一编 人类出现之前的世界

# 第一章 空间和时间中的地球

  • 第一节 人类时空观念的扩展

    • 巴比伦和中国的天文学家研究星象好几百年，依然认为大地是平坦

    • 希腊人最先意识到世界是球状的

    • 望远镜的发明让人们的思想和人生观巨大进步

    • 希腊的科学发展是哲学家推动的， 除了极少数拥有创造才能的人之外，如阿基米德、希埃鲁等，其他人都太过骄傲自大，不愿意向珠宝匠、五金匠、玻璃匠等普通人请教

    • 哲学家缺少机械技能，而工匠没有哲学知识，所以一直到一千年之后，玻璃与天文学之间才建立起联系。

  • 第二节 宇宙空间中的地球

    • 在相距很远的地方存在着一颗颗能够发光发热的星体，这就是恒星。

    • 在现代天文学中，将可见星区划分为许多类，如黄星、红巨星、红矮星、白矮星、双星、突然爆炸或者消失的星

    • 通过无线电望远镜观察发现，脉冲星每隔几秒就会向外放射无线电波

    • 在黄星中，有一颗恒星与我们的距离比较近，它看起来像是一个火球，这就是太阳。太阳和地球之间的距离是9300万

    • 光每秒传播的距离是18.6万英里，从月球照射到地球大约需要1.3秒，从太阳照射到地球大约需要八分钟.

    • 太阳是一团炙热的气体，98％的物质是氢和氦，剩余的2％是气态的铁及其他 元素。太阳内部的热度是表面热度的几千倍，氢不断地转化成氦，并向外释放能量。其实，太阳的热力来源于氢的裂变，这也是氢弹的原理。

    • 流行学说认为，在很久之前，许多浮动在星际间的气体和尘埃在引力的影响下，逐渐聚集在一起，变得越来越致密，慢慢形成了一个中心球体，也就是太阳，还有一些行星及围绕着行星运行的

  • 第三节 地球存在的时间

    • 要意识到地球最终会消失，虽然这是很久以后的事情

    • 几十年前，科学家们都相信太阳和行星在不断地冷却。根据这种说法可知，它们开始于燃烧着的旋转物质，已经失去了很多热量，旋转的速度越来越慢，而且会一直衰弱下去；总有一天，一个白天会像一年的时间一样长，冷却中的太阳会丧失光芒，一动不动地悬挂在空中，地球上的所有生命都会被冻死。

    • 最近，科学研究打破了这种猜测。虽然太阳比以前冷了一些，但它内部的放射性变化会让它的热度持续很长时间。太阳含有大量的氢，以现有的燃烧速度能够燃烧几十亿年。当地球上的生命终结时，与上述预测恰好相反。太阳含有的物质转化成一种新的原子，而且会变得更大更热，将附近的行星都吞没，其中就包括地球。这样一来，我们的地球会被烧掉，而不是被冻死。

  • 第四节 众星中是否有其他世界

    • 只是我们无法测量 我相信一定还有 我相信外星人的存在

  • 体会

    • 从地球的角度去看待自己的生活，感觉坦然了不少。果然知识就是力量啊。

    • 希腊的科学发展时哲学家推进的。哲学家的骄傲阻碍了科学的发展。不同的学科直接的结合才是现在应该做的。斜杠青年一定要做起来啊。

    • 技术不应该人为地强行分割。以后学习的时候，如果这个工具能够解决自己的疑问，那么不论什么学科都要学习。不强行分割，自然而然地我们会变成一个更完整的人

    • 人终有一死，要让活着的时候精彩。
